## Run the project

`yarn`

`yarn start`


Install dependencies in package: `cd packages/name && yarn add -W dependencie-name`

## Storybook for Design System

`yarn story`

## Tests

`yarn test` to run ALL packages test at the same time or `cd packages/<package-name> && yarn test`
